# Gym API

## Install Dependencies and up Image
```
npm install
```
```
 docker-compose up -d
```
## Run dev
```
npm run dev
```

## Build Project
```
npm run build
```
## Run build Project
```
npm run start
```

# Prisma

For dev
```
 npx prisma migrate dev
```
or prod
```
 npx prisma migrate deploy
```
Then Generate prisma client types 
```
  npx prisma generate
```
View data in Prisma Studio
```
npx prisma studio
```
